# PCB-Design-proteus

Embedded System PCB Design (Arduino Based)

This project presents the design and implementation of a Printed Circuit Board (PCB) for a simple embedded system using Arduino and an optical sensor.
The objective is to go through the complete hardware design workflow, from the electronic schematic to the final 3D visualization of the PCB.

The project was developed using Proteus Design Suite, which allows schematic capture, PCB layout, routing, and 3D rendering.

🎯 Project Objectives

Design the electronic schematic of an embedded system

Convert the schematic into a PCB layout

Assign appropriate component footprints (packages)

Route electrical connections on the board

Create a ground plane for better electrical stability

Generate a 3D visualization of the final PCB

🧩 System Components

The system integrates several electronic components commonly used in embedded applications:

 LED – visual indicator

 CNY70 Optical Sensor – used for object detection or line sensing

 Servo Motor – controlled using PWM signals

 Arduino Interface Connector – interface with the microcontroller

 Battery Connector – power supply input

 Resistors – current limiting and signal conditioning
